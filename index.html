Girls Code platforması necə yarandı?

Girls Code platforması Azərbaycanda yaradılmış qadınların İT sektorunda inkişaf etməsi, uğurlu karyera addımları atmağını məqsəd qoymuş platformadır. 10.01.2018 tarixindən etibarən İT sektorunda olan xanımlar ilə birlikdə bu sahədə olan xanımlar üçün müxtəlif seminarlar, görüşlər, vebinarlar keçirməyə başladıq. Fəaliyyətimiz bu period ərzində daha çox online olmaqla davam etmişdir. Təşkil etdiyimiz seminarlar, təlimlər, vebinarlar ərzində 1000-dən çox xanım bizim platformamıza üzv olmuşdur. 2019-cu ildən başlayaraq, online olan fəaliyyətimizi həm də offline rejimə keçirərək, ”Girls Code” adı altında fəaliyyət göstərməyə başladıq.

Məqsədimiz nədir ?

Məqsədimiz İT sektorunda olan bərabərsizliyi aradan qaldırmaq, bu sahədə olan xanımların sayını daha da artırmaqdır. Azərbaycanda qadınların İKT üzrə addımlamasına kömək etmək, dəstək olmaqdır. Biz- Girls Code komandası olaraq, qadınların İT –də səsinin eşidilməsi, qəlblərindəki səsin ətrafda olan insanlar tərəfindən əngəllənməməsi, arzularının arxasınca qaçaraq hərəkət etmələri və ən əsası düşməkdən qorxduqları yerdə bəlkə də, uçacaqlarına inandıracaq iş görürük. Əsas olan isə gördüyümüz işi məhz sevərək, ürəyimizi qoyaraq, böyük tutqu ilə etməyimizdir.Bəzən çox yoruluruq, amma etdiyimiz işə görə aldığımız təşəkkürlər, xanımların məmnuniyyəti və İT –də addımladıqlarını görmək çəkdiyimiz əziyyətdən daha çox enerji almağımızı və yenidən, yenidən və yenidən bütün qüvvəmizi toplayaraq işə başlamağımızı təmin edir!

Girls Code Azerbaijan platforması ilə dəfələrlə müxtəlif layihələrdə tərəfdaşlıq etmişik. Mən hər bir tədbirdə Girls Code komandasının yaratdığı enerji, çəkdiyi zəhmətin hədər getməməsinin, qadınların karyerasına təsirinin şahidi olmuşam. Hər bir tədbirdən sonra iştirakçıların bu platforma ilə bağlı rəylərini müşahidə etmişəm. Əminəm ki, bu layihənin səsi yüksəklərdən gələcək. Azərbaycan qadınını İT sahəsində yüksəlişini təmin edən yeganə platforma olacaqsınız. Ölkəmizdə İT sektorunda xanımların inkişafı üçün çalışan belə bir platformanın mövcudluğu ilə fəxr edirəm.

Gördüyümüz işlərin qarşılığında bu kimi rəylərin gəlməsi bizi gördüyümüz işi daha ürəklənərək görməyə sövq edir.
